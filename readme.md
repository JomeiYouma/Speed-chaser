
# Speed Chaser

## English

### Game Description
Speed Chaser is a fast-paced 3D arcade game where you control a vehicle on an endless road. Your goal is to survive as long as possible, avoid obstacles, and achieve the highest score. The game features a leaderboard using Firebase to store highscores.

### Controls
- **← / Q** and **→ / D**: Move left/right
- **SPACE**: Change color
- **P**: Pause the game
- **S**: Toggle sound

### Technologies Used
- [Three.js](https://threejs.org/) for 3D rendering
- [Firebase](https://firebase.google.com/) for realtime database and highscores
- [Vite](https://vitejs.dev/) for development and build tooling
- Vanilla JavaScript, HTML, CSS

## Français

### Description du jeu
Speed Chaser est un jeu d'arcade 3D rapide où vous contrôlez un véhicule sur une route sans fin. Votre objectif est de survivre le plus longtemps possible, d'éviter les obstacles et d'obtenir le meilleur score. Un classement en ligne est disponible grâce à Firebase.

### Contrôles
- **← / Q** et **→ / D** : Déplacer à gauche/droite
- **ESPACE** : Changer de couleur
- **P** : Mettre le jeu en pause
- **S** : Activer/désactiver le son

### Technologies utilisées
- [Three.js](https://threejs.org/) pour le rendu 3D
- [Firebase](https://firebase.google.com/) pour la base de données et les scores
- [Vite](https://vitejs.dev/) pour le développement et le build
- JavaScript, HTML, CSS
