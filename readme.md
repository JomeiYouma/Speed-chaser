### 🏎️ Speed Chaser

Un jeu d'arcade frénétique développé avec **Three.js** et **Firebase**. Pilotez un vaisseau spatial à travers des murs colorés, changez de couleur en un éclair et survivez le plus longtemps possible à une vitesse croissante.

---

## 🇫🇷 Français

### 📝 Description
**Speed Chaser** est un runner infini en 3D où les réflexes sont primordiaux. Le joueur contrôle un vaisseau sur trois lignes (lanes) et doit traverser des murs qui s'approchent de plus en plus vite. 

La particularité ? Chaque mur a une couleur spécifique. Le vaisseau doit impérativement correspondre à cette couleur pour passer au travers sans s'écraser.

### 🎮 Commandes
* **Q / D** ou **Flèches Gauche/Droite** : Changer de ligne (Gauche, Milieu, Droite).
* **Espace** ou **Clic Souris** : Changer la couleur du vaisseau (Cycle entre les couleurs disponibles).

### 🛠️ Fonctionnalités
* **Difficulté Progressive** : La vitesse des murs augmente au fil du temps.
* **Système de Score** : Plus vous survivez, plus votre score grimpe.
* **Leaderboard Firebase** : Sauvegardez votre meilleur score et comparez-le avec les autres joueurs.
* **Graphismes 3D** : Environnement immersif utilisant Three.js.

### 🚀 Installation
1. Clonez le dépôt : `git clone https://github.com/votre-utilisateur/speed-chaser.git`
2. Installez les dépendances : `npm install`
3. Configurez vos clés Firebase dans votre fichier de configuration.
4. Lancez le projet : `npm run dev`

## 🇬🇧 English

### 📝 Description
**Speed Chaser** is a fast-paced 3D infinite runner where reflexes are key. Players control a spaceship across three lanes and must pass through approaching walls at high speed. 

The catch? Each wall has a specific color. The ship must match that color to pass through safely instead of crashing.

### 🎮 Controls
* **A / D** (or **Q / D**) or **Left/Right Arrows**: Change lanes (Left, Center, Right).
* **Spacebar** or **Mouse Click**: Toggle ship color (Cycle through available colors).

### 🛠️ Features
* **Increasing Speed**: Difficulty ramps up exponentially over time.
* **Scoring System**: The longer you survive, the higher your score.
* **Firebase Leaderboard**: Save your high scores to the database and compete globally.
* **3D Graphics**: Immersive environment powered by Three.js.

### 🚀 Installation
1. Clone the repository: `git clone https://github.com/your-username/speed-chaser.git`
2. Install dependencies: `npm install`
3. Set up your Firebase credentials in your config file.
4. Start the project: `npm run dev`
